## TP 12 : Gestion des fichiers

## Exercice 1 : FileReader/FileWriter, BufferedReader/BufferedWriter, Files

**Objectifs pédagogiques**
Maîtriser les opérations de base sur les fichiers : lecture, écriture, copie, et suppression.
Utiliser les flux (FileReader/FileWriter, BufferedReader/BufferedWriter, Files API).
Gérer les formats courants : texte brut, CSV, objets sérialisés.
Appliquer le try-with-resources pour la fermeture automatique des ressources.

## Exercice 2 : PrintWriter et BufferedReader

**Objectifs pédagogiques**
Découvrir l’usage de la classe PrintWriter pour écrire dans un fichier texte avec formatage simple.
Combiner PrintWriter et BufferedReader pour écrire, lire et mettre à jour des fichiers.
Appliquer le try-with-resources pour garantir la fermeture des flux.
